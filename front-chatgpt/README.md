# Frontend ChatGPT Clone

Frontend similar a ChatGPT con integración de WhatsApp y Quivr.

## Características

- Panel izquierdo con gestión de conversaciones (últimas 10)
- Chat central con integración a Quivr
- Panel derecho con controles:
  - Cambio de tema (oscuro/claro)
  - Conexión de WhatsApp con QR embebido
  - Actualización de Brain

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```
